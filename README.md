## 📬 Resurs taklif qilish

Foydali sayt yoki ilovani katalogga qo'shish uchun:

1. Saytga o'ting: [elink.uz](https://elink.uz)
2. **"Resurs taklif qilish"** tugmasini bosing
3. Quyidagi ma'lumotlarni kiriting:
   - Sayt / Ilova nomi
   - URL manzil
   - Qisqacha tavsif
   - Telegram yoki telefon (ixtiyoriy)
4. **"Taklif qilish"** tugmasini bosing

> Taklifingiz ko'rib chiqilgach, ma'qullansa katalogga qo'shiladi.

---

## 🐛 Muammo bildirish

Agar biror resurs ishlamasa:

1. Resurs sahifasini oching
2. **"Muammo bildirish"** bo'limiga o'ting
3. Muammo turini tanlang:
   - 🔴 Sayt ochilmayapti
   - 🐢 Sekin ishlayapti
   - 🔗 Xato URL
   - ❓ Boshqa muammo
4. Izoh qoldiring va yuboring
